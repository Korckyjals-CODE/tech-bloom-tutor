**üß© 3. Core Learning Section**

**üîç Mini Lesson: "What does it mean to install Python?"**

1.  **Python** is a *programming language* --- software that allows you
    to write code and make the computer follow instructions.

2.  To use it, you must first **install it**, meaning you copy the
    Python program onto your computer.

3.  Once installed, you can open a tool called **Windows Terminal** (a
    black window where you type commands).

4.  You can check if Python is installed by typing:

5.  python \--version

If Python is installed, this shows its version number.

**üß≠ Key Terms**

  -----------------------------------------------------------------------
  **Term**        **Meaning (student-friendly)**
  --------------- -------------------------------------------------------
  **Python**      A programming language used to create software, games,
                  and automation.

  **Installer**   A file that adds Python to your computer.

  **PATH**        A setting that helps the computer find Python from any
                  folder.

  **Windows       A window where you can type commands instead of
  Terminal**      clicking icons.

  **Version**     The specific release number of Python you have
                  installed.
  -----------------------------------------------------------------------
